# React ecommerce online store application


## General Info 

front-end: Reactjs, Redux, Axios, Sass, react-bootstrap

back-end: Node.js, Express, Restful API, paypal-sdk, mongodb, mongoose, jwt(jsonwebtoken), heroku, firebase, swaggerHub
